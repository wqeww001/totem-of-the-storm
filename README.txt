My first Minecraft mod for Forge 1.20.1. Adds a Storm Totem — a structure that summons lightning and gives buffs.

> ⚠️ I'm still learning! Textures are bad, there may be bugs. Please don't judge too harshly 🙏

---

## ⚡ What the mod does

- Build a totem from 3 blocks (2 Storm Blocks + 1 Rod)
- Activate with Storm Flint
- Lightning, particles, thunder sound
- Player gets Speed and Jump Boost
- Nearby crops instantly grow
- Animals glow and drop experience

---

## 🛠️ Crafting

| Item | Ingredients |
|------|-------------|
| Charged Copper Ingot | Copper + Redstone + Glowstone Dust |
| Storm Infused Block | Charged Ingots + Smooth Stone |
| Storm Rod | Charged Ingots + Storm Block |
| Storm Dust | Glowstone Dust + Redstone + Gunpowder |
| Storm Flint | Storm Dust + Charged Ingots + Flint & Steel |
