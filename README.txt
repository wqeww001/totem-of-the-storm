## ✨ Overview

**Totem of the Storm** is a lightweight gameplay mod focused on a single powerful mechanic:  
a structure that summons lightning and enhances the world around it.

It’s designed to feel *vanilla-friendly*, but with a dramatic twist.

---

## ⚡ Features

- ⚙️ Simple multi-block structure  
- ⚡ Lightning summoning system  
- 💨 Player buffs (Speed, Jump Boost)  
- 🌱 Instant crop growth  
- 🐾 Mob interaction (glow + XP drops)  
- 🌩️ Atmospheric particles & sound
