## ⚡ About

**Totem of the Storm** is my first Minecraft Forge mod.  
It adds a mystical structure that channels lightning and grants powerful buffs.

> ⚠️ This is an early project — expect bugs and rough textures.

---

## ⚡ Features

- Build a totem from 3 blocks  
- Activate it using a special item  
- Summons lightning and storm effects  
- Grants **Speed** and **Jump Boost**  
- Instantly grows nearby crops  
- Animals glow and drop experience  

---

## 🛠️ Crafting

| Item | Recipe |
|------|--------|
| **Charged Copper Ingot** | Copper + Redstone + Glowstone Dust |
| **Storm Infused Block** | Charged Ingots + Smooth Stone |
| **Storm Rod** | Charged Ingots + Storm Block |
| **Storm Dust** | Glowstone Dust + Redstone + Gunpowder |
| **Storm Flint** | Storm Dust + Charged Ingots + Flint & Steel |
